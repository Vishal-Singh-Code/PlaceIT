# PlaceIT Exam SaaS

This is a Next.js App Router project for running multi-exam, multi-test mock assessments with analytics and leaderboards.

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Set the `MONGODB_URI` environment variable in a `.env.local` file:

```bash
MONGODB_URI=mongodb://localhost:27017/placeit
```

3. Run the development server:

```bash
npm run dev
```

Then open `http://localhost:3000` in your browser.
